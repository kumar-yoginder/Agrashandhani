## 🔍 ERROR ROOT CAUSES

### Why 401 Errors Occur
```
401 = Unauthorized
Root Cause: Missing API key in .env file
Example: VT_API_KEY not set
Fix: Add to .env and restart
```

### Why 403 Errors Occur
```
403 = Forbidden  
Root Cause: Invalid, expired, or revoked API key
Example: Shodan key expired or has no permissions
Fix: Generate new key or verify permissions
```

### Why 404 Errors Occur (Expected)
```
404 = Not Found
Root Cause: Data doesn't exist in database
Example: Random IP not in Cymru's hash database
Fix: None - this is normal for some IOCs
```

### Why Timeouts Occur
```
timeout = Read Timeout
Root Cause: API slower than 10 seconds
Example: OTX API is slow tonight
Fix: Increase timeout or retry with backoff
```
